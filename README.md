# UWISpots
A crowdspurced location-based web app for university students to track useful spots around campus. 
## Features
- Interactive map showing pins for study spots, food vendors etc.
- Users can drop new pins and write reviews
- Upvote/downvote existing pins
- User authentication (login/signup)

## Tech Stack 
- HTML, CSS, Javascript
- Firebase (Firestore + Authentication)
- Google Maps API
- Hosted on Netlify
